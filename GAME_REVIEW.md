### The two shortlist systems (structural issue)

The app has two parallel "shortlist" concepts:
1. **`_shortlists` table** — powers tick-boxes in Roster/Radar/Shortlist tabs (new)
2. **`_matches.status = 'shortlisted'`** — used by IntrosTab display + game XP counts (old)

No recruiter-facing UI currently changes `_matches.status`. The "Scouting XP" formula (`shortlisted * 50`) is counting AI pipeline statuses, not recruiter ticks. **The shortlist quest is uncompletable through normal use.**

Fix options (ascending complexity):
- **A (quickest):** Change `useGameStats` to fetch `_shortlists` count directly: `GET /api/shortlists/count?recruiter=X`. Wire as `shortlistXP = shortlistCount * 50`. 
- **B (cleaner long-term):** Unify both shortlist tables — when tick-box fires, also set `_matches.status = 'shortlisted'`. Keeps data model consistent but requires backend change.

### UX gaps (not yet fixed)

**Level-up celebration is missing.** The rank badge changes color and text silently when you advance. This is the single highest-impact moment in any progression system. Suggested: detect `level.level !== prevLevel.level` in `GameDashboard`, show a full-screen overlay (dark navy + light rays + large rank badge + "RANK UP!" + particle burst) for 3 seconds.

**Leaderboard is missing.** A team of recruiters with XP and no scoreboard is a missed opportunity. All recruiter data is in `_recruiters` + `_role` — computable server-side. Add to `PlayerTab` as a ranked list.

**Daily objectives don't exist.** All 5 quests are permanent one-time unlocks. Once done, game mode is inert. Suggested: seed 3 daily tasks from `Date.now()` (e.g., "Shortlist 2 candidates today", "Open Radar for any role"), grant 2× XP for completion.

**Functional tabs lose the game skin.** Going from Home → Missions is a visual context switch from dark navy/gold → light mode. Low priority to fix fully (high effort), but could partially solve with a dark header band and XP hint on action buttons in game mode.

**ProfileTab (Player) is completely ungamified.** It's a plain form. Should show at minimum: rank badge, total XP, achievements earned, and position in team leaderboard.
